# RF Remote user manual

### Short Press 
- A --> Increment score Team A
- B --> Increment score Team B
- C --> Clear current match score
- D --> Toggle display Mode Tittle/Score/Timer
### Long Press 
- C --> Clear all score from display
- D --> Toggle service side Left/Right
